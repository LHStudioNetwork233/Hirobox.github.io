Dear user, welcome to Hirobox Cilent!

Hirobox is an auxiliary tool for modifying Minecraft: PE.

Mouse support

Toolbox supports mouse & keyboard play on Android 8 and above.

Device support

The toolbox only supports Android 8 to Android 10, and a version that is too high or too low will cause errors.

storage directory

See the files in the compressed package for details.

Skin loading

The toolbox has the ability to load skin packs with custom geometries. Use an external storage location and put the skin packs into the internal storage at games/com.mojang/minecraftpe/skin_packs

Texture pack resources

The toolbox can load custom materials. Please use an external storage location, unzip the resource pack into a folder, and put it into the internal storage of games/com.mojang/resource_packs/

Resource pack skipped

You can skip mandatory resource packs when joining a server using the toolbox.

Configuration issues

Basically the configuration is placed in json, you can also adjust the configuration yourself
